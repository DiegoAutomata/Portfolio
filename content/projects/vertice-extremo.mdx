## Resumen del Proyecto

**Vértice Extremo** es una plataforma web premium diseñada para una agencia de turismo de aventura especializada en trekking y montañismo de alta dificultad en la Patagonia (El Chaltén, Fitz Roy, etc.) y expediciones internacionales (Everest Base Camp, Mont Blanc). 

La interfaz rompe con el diseño web tradicional, proponiendo una estética inmersiva de alto impacto visual orientada al público aventurero, con tipografías audaces, contrastes intensos, transiciones fluidas de imágenes basadas en desplazamiento y un mapa interactivo para localizar los campamentos base.

## Características Principales

- **Efectos de Desplazamiento Parallax**: Experiencia cinemática en la página de inicio que simula la profundidad de la montaña a medida que el usuario realiza scroll, acompañada de elementos flotantes tridimensionales.
- **Galería Dinámica de Momentos**: Visor interactivo y lightbox personalizado con transiciones fluidas para fotografías en alta resolución, simulando una exposición fotográfica física de la naturaleza patagónica.
- **Detalle de Expediciones**: Páginas dedicadas por expedición que muestran el nivel de dificultad, itinerarios, equipamiento recomendado y un botón de reserva integrado.
- **Mapa Interactivo Integrado**: Geolocalización del Base Camp principal en El Chaltén mediante un mapa interactivo personalizado que emula la estética oscura y monocromática del sitio.
- **Autenticación con Supabase**: Sistema de login y registro de usuarios integrado directamente con Supabase Auth y base de datos relacional para gestionar perfiles de montañistas y reservas de viajes de manera segura.

## Stack Tecnológico

- **Frontend**: React, Vite, Tailwind CSS y TypeScript.
- **Animaciones**: Framer Motion de alto rendimiento para gestos, scrolls paralelos y efectos de transición inmersivos.
- **Backend / DB / Auth**: Supabase (PostgreSQL + RLS + JWT Auth) para almacenamiento persistente y autenticación sin servidor.
- **Iconografía**: Lucide React para elementos visuales de control y estado.
- **Despliegue**: Vercel con integración automática y optimización de entrega de contenidos estáticos y dinámicos.
