## Overview

Vosstudio is a conversion-optimized recruitment landing page built with Next.js 16 and a cyberpunk/neon design system. It combines SEO-first architecture with an aggressive lead generation funnel that channels all prospects through WhatsApp.

## Technical Highlights

- **Cyberpunk Design System**: Dark theme with `#09090b` background, `#FF007D` neon pink, and `#00F0FF` neon cyan, backed by comprehensive design tokens in `DESIGN.md`.
- **SEO-First Architecture**: Full metadata on every page (title, description, keywords, OpenGraph, Twitter cards), `sitemap.ts`, `robots.ts`, and JSON-LD structured data for Organization schema.
- **Multi-Language Support**: Google Translate integration with auto-detection of browser language, supporting 8 languages (ES, EN, PT, FR, DE, IT, RO, RU) via a custom `LanguageSwitcher` component.
- **WhatsApp-Driven Funnel**: No backend form submission — the entire application flow redirects to WhatsApp with pre-filled messages. Floating WhatsApp button persists across all pages.
- **Conversion-Optimized UX**: Hero with glow effects, urgency/authority bars, benefit grid, 3-step onboarding flow, and dual CTA buttons with neon glow shadows.
- **Next.js 16 App Router**: Built on the latest Next.js with experimental MCP server support and full RSC architecture.
